# TravelGenie AI

TravelGenie AI is an AI-powered eco-tourism and homestay discovery platform designed to help travelers plan personalized trips across Uttarakhand.

## Features
- Personalized itinerary generation
- Budget-based travel planning
- Homestay recommendations
- Weather-aware travel suggestions
- Local attractions and activity discovery

## Tech Stack
### Frontend
- React JS

### Backend
- Python FastAPI

### Database
- PostgreSQL (Supabase)

## Future AI Features
- Budget Agent
- Weather Agent
- Accommodation Agent
- Activity Agent
- Local Experience Agent
